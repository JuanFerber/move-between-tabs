# Navegador de Pestañas (Tab Navigator)

Esta extensión de Chrome está diseñada para mejorar tu productividad permitiéndote gestionar tus pestañas utilizando atajos de teclado rápidos y eficientes. Olvídate de usar el mouse para tareas repetitivas de navegación.

## Características Principales

La extensión proporciona 4 comandos clave con atajos de teclado predeterminados para una gestión fluida de las pestañas:

*   **Mover a la pestaña anterior (Izquierda):** Navega cíclicamente hacia la izquierda entre tus pestañas abiertas.
    *   Comando: `switch-left`
    *   Atajo: `Alt` + `Shift` + `Left` (Flecha Izquierda)
*   **Mover a la pestaña siguiente (Derecha):** Navega cíclicamente hacia la derecha.
    *   Comando: `switch-right`
    *   Atajo: `Alt` + `Shift` + `Right` (Flecha Derecha)
*   **Abrir y enfocar nueva pestaña:** Abre inmediatamente una nueva pestaña y coloca el foco en ella.
    *   Comando: `open-and-focus-new-tab`
    *   Atajo: `Alt` + `Shift` + `Up` (Flecha Arriba)
*   **Cerrar pestaña actual:** Cierra rápidamente la pestaña que estás viendo.
    *   Comando: `close-actual-tab`
    *   Atajo: `Alt` + `Shift` + `Down` (Flecha Abajo)

## Instalación (Modo Desarrollador)

1.  Clona o descarga este repositorio.
2.  Abre Chrome y ve a `chrome://extensions/`.
3.  Activa el "Modo de desarrollador" (Developer mode) en la esquina superior derecha.
4.  Haz clic en "Cargar descomprimida" (Load unpacked).
5.  Selecciona la carpeta de este proyecto.
